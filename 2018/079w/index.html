动态规划 (Dynamic Programming)¶

  • DP 的相关内容
  • 以 DP 为例, 探讨一下, 如何迅速学习一个新算法

DP 有什么用¶

A : "1+1+1+1+1+1+1+1 =?" 
A : "上面等式的值是多少"
B : 计算 "8!"
A : 在上面等式的左边写上 "1+" 
A : "此时等式的值为多少"
B : quickly "9!"

A : "你怎么这么快就知道答案了"
A : "只要在8的基础上加1就行了"
A : "所以你不用重新计算因为你记住了第一个等式的值为8!动态规划算法也可以说是 '记住求过的解来节省时间'"
  • DP 的核心就是记住已经解决过的子问题的解
  • 很简单, 但是超有用

超简单例子 (Fibonacci)¶

fib(1), fib(2) = 1, 1

实际运用¶

  • 其实就是对递归的优化, 本质是种思路
  • 总会有一个表, 最终结果可以通过查表直接得到
  • 这种思维不止是算法, 而是到处都能用到
    • 建表
    • 查表
    • 递归
