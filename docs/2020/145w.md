### 熊本 -> 珠海怼会感想
- 线下的好处被高看
    + 本以为线下聚会有利于推动俺扣主题
    + 结果线下变化要素太多,俺紧张地讲不出来
    + 缺乏 slides 的帮助,真是太吃力了...
- 线上的好处被低估
    + 以前只觉得线上麻烦
        * 断续的网络信号
        * slides 的准备
        * 长达 1 个半小时的会议
    + 经历线下后,大感线上之靠谱
        * 无通勤时间
        * 可投影 slides
        * 更多人参加...
- 聚会的意义
    + 触发有趣之主题
        * 一场线下触发了俺4 个以上的想法
        * 线上能触发俺 1 个想法都非常不错
        * 不过长期来看,1 周 1 个想法也足够了
        * 关键是,咋「有种」得把想法做出来...
            * 所以启动了个人分支的 logging
            * 记录想法/主题
            * 汇集有趣主题下的资料,当素材来使...
