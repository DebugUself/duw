暗渠1:主动¶

  • 珍教建议

    • 感觉主动一下挺重要的,很多时候,觉得自己明明资格还差一截,主动之后就有机会了.
    • 不知道怎么主动,可以从表达感谢开始. 真诚的感谢别人对你的帮助,有时候会有意想不到的收获.
    • 作品 + 混个脸熟,很多时候机会就来了.

  • 珍教案例

    • Udacity Code Reviewer
      • 在没有公开招募,名额有限情况下,积极联系导师并努力完成自己作品,最终争取到reviewer的名额,不仅帮助其他同学review code,还赚到美元.
    • Py103
      • 在带几个月大孩子+上班的忙碌过程中,依然坚持教练.

@zoejane - 小搜索能搞定大问题¶

  • 周会分享链接

  • 背景

    • 妈妈因为一件比较重要的事情,需要联系到一位很久没联络的朋友
    • 她自己尝试了网络搜索,包括向多位朋友打听,最终无果. 她甚至打算去一趟外地,但是去外地也不一定会解决这个问题. 这件事情困扰了她大半个月了.
  • 突破:20分钟搜索居然就搞定了问题.
    • 尝试搜索. 首先是 Google 人名,搜到了他很早之前发布的一则招聘信息. 但是这个店已经不在了,手机号也不是本人在用了. 这是也是我妈妈自己也找到过的信息.
    • 根据这则招聘信息以及我妈妈的描述,我推测他很可能是自己开店,很有可能是个体工商户或者企业法人,于是我通过搜索企业信用的网站,搜索法人名字,搜到了几家公司.
    • 其中两家公司和我妈妈描述的对方所在省相符,推测很有可能是他开办的. 我选择他最近注册的一家企业查看详情,了解到他有和另外两个人合伙开店,同时我记录下店名和地址.
    • 再次 Google 企业所在城市 + 店名中的关键词,碰巧搜到了他曾经一个合伙人的电话.
    • 既然合伙办过企业,应该很大可能性现在还有联系. 于是我把这个合伙人的电话告诉我妈妈,我妈妈和对方说明来意后,顺利拿到了她想找的人的电话.
    • 整个搜索过程加联系过程全程花费时间20分钟左右. 我妈妈都感觉非常惊讶.
  • 感悟
    • 没想到通过小小的搜索, 就搞定了我妈妈各种联系朋友,甚至要跑去外地都搞定不了的问题. 互联网中蕴含的资源真的很多,就看自己会不会去发掘.
    • 以前也遇到过很多东西搜了半天搜不到什么有用信息的情况. 这次体验让我感受到很多东西不是搜不到,而是搜索功力不到家,很多东西可能是要通过别的相关线索,结合一些推理去进一步发掘.
    • 用自己的搜索能力搞定生活中的大问题的感觉太开心了. 用真实的问题而不是教科书式的问题驱动自己,能够得到更多的正反馈和动力.
